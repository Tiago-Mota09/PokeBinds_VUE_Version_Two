# PokeBinds_Two

## Sobre o Projeto:
Teste para Binds versão 2. Sistema conectado com a API PokeApi que exibe lista completa de Pokémons ou lista através de busca/filtro pré-selecionado pelo usuário.

## 🛠 Tecnologias

- VueJS
- Axios
